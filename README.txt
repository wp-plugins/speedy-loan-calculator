== Description ==

Speedy Loan Calculator is a WordPress widget that puts a simple loan calculator in your widget area. It features immediate calculation, and allows the user to change the variables (loan term, amount, and rate).

Features:

*   Simple 3-input interface
*   Allows user to choose rate, loan term, and amount
*   Reset button
*   Can place in any widget location

Pro version: [Speedy Loan Calculator for WordPress](http://gavinr.com/speedy-loan-calculator) features configurable presets and support.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload zip on the WordPress Plugins page
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Appearance > Plugins` and add the widget to a widget area.
1. Set the title, save, and start using!

== Frequently Asked Questions ==

= Are all three inputs modifyable? =

Yes, the user can change the loan amount, length, or rate.

= What about presents? =

You can set the presets in the pro version of the plugin, offered at [Speedy Loan Calculator for WordPress](http://gavinr.com/speedy-loan-calculator)
